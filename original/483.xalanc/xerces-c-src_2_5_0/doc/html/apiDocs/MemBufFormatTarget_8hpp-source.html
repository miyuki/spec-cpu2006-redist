00091 /*
00092  * The MemBufFormatTarget is a derivative from XMLFormatTarget, which user code
00093  * may plug into DOMWriter to retrieve the serialized XML stream (from DOM Tree)
00094  * in a memory buffer.
00095  *
00096  * The MemBufFormatTarget is initalized to have a memory buffer of 1023 upon
00097  * construction, which grows as needed. The buffer will be deleted when
00098  * MemBufFormatTarget is destructed; or will be reset when the reset() function
00099  * is called.
00100  *
00101  * The MemBufFormatTarget returns a NULL terminated XMLByte stream upon request,
00102  * through the method getRawBuffer(), and user should make its own copy of the
00103  * returned buffer if it intends to keep it independent on the state of the
00104  * MemBufFormatTarget.
00105  */
00106
