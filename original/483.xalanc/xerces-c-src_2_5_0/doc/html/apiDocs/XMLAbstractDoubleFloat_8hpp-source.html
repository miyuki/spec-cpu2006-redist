00141 /***
00142  * 3.2.5.1 Lexical representation
00143  *
00144  *   double values have a lexical representation consisting of a mantissa followed,
00145  *   optionally, by the character "E" or "e", followed by an exponent.
00146  *
00147  *   The exponent ·must· be an integer.
00148  *   The mantissa must be a decimal number.
00149  *   The representations for exponent and mantissa must follow the lexical rules
00150  *   for integer and decimal.
00151  *
00152  *   If the "E" or "e" and the following exponent are omitted,
00153  *   an exponent value of 0 is assumed.
00154 ***/
00155 
00156 /***
00157  * 3.2.4.1 Lexical representation
00158  *
00159  *   float values have a lexical representation consisting of a mantissa followed,
00160  *   optionally, by the character "E" or "e", followed by an exponent.
00161  *
00162  *   The exponent ·must· be an integer.
00163  *   The mantissa must be a decimal number.
00164  *   The representations for exponent and mantissa must follow the lexical rules
00165  *   for integer and decimal.
00166  *
00167  *   If the "E" or "e" and the following exponent are omitted,
00168  *   an exponent value of 0 is assumed.
00169 ***/
