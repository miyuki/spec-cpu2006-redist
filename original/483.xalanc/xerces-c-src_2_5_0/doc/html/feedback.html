Questions or Comments
 

Please browse through this bundled documentation completely. Most of the common questions have been answered in the FAQ's. Specifically, do read the answer to " Is there any kind of support available for Xerces-C++?". Browsing this documentation, may be the quickest way to get an answer. Of course, if all else fails, as mentioned in the link above, you can post a question to the Xerces-C++ mailing list.

See Bug Reporting if you would like to report a defect (greatly appreciated!).
